## Guardrails

- Ne pas mélanger OptiAI et Next Move dans le positionnement.
- Ne pas exposer d’information client ou confidentielle.
- Garder les fiches comme preuves d’expertise et supports d’apprentissage, pas comme journal intime.
- Les projets personnels/spirituels restent dans une catégorie distincte des cas business.

## Vérification

Prochaine action recommandée : exécuter `npm run build` sur la branche puis reviewer le rendu `/projects`.
